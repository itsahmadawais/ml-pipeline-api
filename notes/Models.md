# Models (When to use what)
There are different models for different purposes:

Numbers → Regression models → predict value

Categories → Classification models → predict label

Groups → Clustering → find patterns

Text meaning → NLP models → understand language

Similarity → Embeddings → compare meaning

## 1. Regression Models (used to predict a number)
- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor
- XGBoost / Gradient Boosting

## 2. Classification Models (predict a category)
- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier
- XGBoost Classifier

## 3. Clustering Models (no labels)
- K-Means
- DBSCAN

## 4. NLP / Text Models
- Embeddings (Word2Vec / Open AI embeddings)
- Transformers (BERT, GPT-style models)

## 5. Recommendation / Ranking Models
- Consine Similarity
- Learning to Rank
